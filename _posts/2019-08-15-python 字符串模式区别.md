python 字符串模式 u, r, b 的区别。



## 操作

```python
# u: 表示unicode字符串，默认模式，里边的特殊字符会被识别。
print(u'hi\thi\thi')  # hi	hi	hi

# b: 表示二进制字符串，括号内的内容原样输出。
print(b'hi\thi\thi')  # b'hi\thi\thi'

# r：不转义字符串，要输出的内容原样输出。
print(r'hi\thi\thi')  # hi\thi\thi
```
